# ray-tracing-fs

Ray Tracing in Fragment Shader

## Dependency
- Next.js
- Three.js
- React Three Fiber

## Running Development Server
```console
$ npm i
$ run npm dev
```

## Shaders
The shader files are in `/shaders`.
- [Fragment Shader](/shaders/ray-tracing-fragment.glsl)
- [Vertex Shader](/shaders/ray-tracing-vertex.glsl)
